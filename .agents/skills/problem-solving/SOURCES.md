# Sources & Attribution

## Framework Origins

### 5 Whys
| Aspect | Source |
|--------|--------|
| Origin | Sakichi Toyoda, Toyota Production System (TPS) |
| Context | Developed as part of Lean manufacturing methodology |
| Reference | Toyota Motor Corporation |

### Fishbone Diagram (Ishikawa Diagram)
| Aspect | Source |
|--------|--------|
| Origin | Kaoru Ishikawa, 1960s |
| Context | Kawasaki shipyards quality management |
| Reference | "Guide to Quality Control" by Kaoru Ishikawa (1968) |
| Also known as | Cause-and-effect diagram, herringbone diagram |

### Root Cause Analysis (RCA)
| Aspect | Source |
|--------|--------|
| Origin | Multiple sources (engineering, medicine, safety) |
| 5 Ms Framework | Toyota Production System |
| Reference | ASQ (American Society for Quality) |

### First Principles Thinking
| Aspect | Source |
|--------|--------|
| Classical origin | Aristotle's philosophy |
| Modern application | Physics, engineering, entrepreneurship |
| Popular reference | Elon Musk's problem-solving approach |
